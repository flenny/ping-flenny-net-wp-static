TCP and UDP ports used by Apple Filing Protocol (AFP)

This weekend I have activated the Apple Filing plugin for my OpenMediaVault server in order to improve the overall experience in accessing data from my MacBook. Due to the fact that the firewall on the OMV server is enabled and drops all incoming connections by default, I had to open the ports used by Apples Filing Protocol manually.

Port Protocol Name Purpose
427 TCP Service Location Protocol (SLP) Network Browser, Authentication
523 TCP Apple Filing Protocol (AFP) Apple File Service
5353 UDP Multicast DNS (MDNS) Service Announcement
